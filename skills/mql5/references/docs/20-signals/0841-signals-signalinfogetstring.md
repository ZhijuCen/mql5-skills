# SignalInfoGetString

Returns the value of [string](/en/docs/basis/types/stringconst) type property of signal copy settings.

```
string  SignalInfoGetString(
   ENUM_SIGNAL_INFO_STRING     property_id,     // property identifier
   );

```

Parameters

property_id

[in]  Signal copy settings property identifier. The value can be one of the values of the [ENUM_SIGNAL_INFO_STRING](/en/docs/constants/tradingconstants/signalproperties#enum_signal_info_string) enumeration.

Return Value

The value of [string](/en/docs/basis/types/stringconst) type property of signal copy settings.
