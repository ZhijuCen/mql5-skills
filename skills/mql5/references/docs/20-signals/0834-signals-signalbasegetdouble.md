# SignalBaseGetDouble

Returns the value of [double](/en/docs/basis/types/double) type property for selected signal.

```
double  SignalBaseGetDouble(
   ENUM_SIGNAL_BASE_DOUBLE     property_id,     // property identifier
   );

```

Parameters

property_id

[in]  Signal property identifier. The value can be one of the values of the [ENUM_SIGNAL_BASE_DOUBLE](/en/docs/constants/tradingconstants/signalproperties#enum_signal_base_double) enumeration.

Return Value

The value of [double](/en/docs/basis/types/double) type property of the selected signal.
