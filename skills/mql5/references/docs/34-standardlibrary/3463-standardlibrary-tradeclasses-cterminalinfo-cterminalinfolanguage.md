# Language

Gets the information about the language of the client terminal.

```
string  Language() const 

```

Return Value

Language used in the client terminal.

Note

The terminal language is defined by [TerminalInfoString()](/en/docs/check/terminalinfostring) function ([TERMINAL_LANGUAGE](/en/docs/constants/environment_state/terminalstatus#enum_terminal_info_integer) property).
