# LineVertical

Draws a segment of a vertical line.

```
void  LineVertical(
   int         x,       // X coordinate
   int         y1,      // Y coordinate
   int         y2,      // Y coordinate
   const uint  clr      // color
   );

```

Parameters

x

[in]  Segment's X coordinate.

y1

[in]  Y coordinate of the segment's first point.

y2

[in]  Y coordinate of the segment's second point.

clr

[in]  Color in ARGB format.
