# Circle

Draws a circle

```
void  Circle(
   int         x,       // X coordinate
   int         y,       // Y coordinate
   int         r,       // radius
   const uint  clr      // color
   );

```

Parameters

x

[in]  X coordinate of the center of the circle.

y

[in]  Y coordinate of the center of the circle.

r

[in]  Circle radius.

clr

[in]  Color in ARGB format.
