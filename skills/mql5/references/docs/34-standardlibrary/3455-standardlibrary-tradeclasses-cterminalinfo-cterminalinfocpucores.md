# CPUCores

Gets the information about the number of CPU cores in the system.

```
int  CPUCores() const 

```

Return Value

Number of CPU cores in the system.

Note

The number of CPU cores is defined by [TerminalInfoInteger()](/en/docs/check/terminalinfointeger) function ([TERMINAL_CPU_CORES](/en/docs/constants/environment_state/terminalstatus#enum_terminal_info_integer) property).
