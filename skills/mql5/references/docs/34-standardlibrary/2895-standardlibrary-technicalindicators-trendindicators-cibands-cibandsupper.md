# Upper

Returns the buffer element of the upper line by the specified index.

```
double  Upper(
   int  index      // index
   )

```

Parameters

index

[in]  Upper line buffer element index.

Return Value

The buffer element of the upper line of the specified index, or [EMPTY_VALUE](/en/docs/constants/namedconstants/otherconstants) if there is no correct data.
