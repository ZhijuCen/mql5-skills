# DataPath

Gets the information about the terminal data folder.

```
string  DataPath() const 

```

Return Value

Data folder of the client terminal.

Note

Client terminal data folder is defined by [TerminalInfoString()](/en/docs/check/terminalinfostring) function ([TERMINAL_DATA_PATH](/en/docs/constants/environment_state/terminalstatus#enum_terminal_info_integer) property).
