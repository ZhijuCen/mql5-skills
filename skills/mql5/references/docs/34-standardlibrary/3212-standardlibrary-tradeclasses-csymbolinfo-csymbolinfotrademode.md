# TradeMode

Gets the order execution type.

```
ENUM_SYMBOL_TRADE_MODE  TradeMode() const

```

Return Value

Order execution type from [ENUM_SYMBOL_TRADE_MODE](/en/docs/constants/environment_state/marketinfoconstants#enum_symbol_trade_mode) enumeration.

Note

The symbol should be selected by [Name](/en/docs/standardlibrary/tradeclasses/csymbolinfo/csymbolinfoname) method.
