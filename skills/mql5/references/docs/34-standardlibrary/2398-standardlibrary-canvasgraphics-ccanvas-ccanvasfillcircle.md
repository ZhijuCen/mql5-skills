# FillCircle

Draws a filled circle.

```
void  FillCircle(
   int         x,       // X coordinate
   int         y,       // Y coordinate
   int         r,       // radius
   const uint  clr      // color
   );

```

Parameters

x

[in]  X coordinate of a filled circle center.

y

[in]  Y coordinate of a filled circle center.

r

[in]  Filled circle radius.

clr

[in]  Color in ARGB format.
