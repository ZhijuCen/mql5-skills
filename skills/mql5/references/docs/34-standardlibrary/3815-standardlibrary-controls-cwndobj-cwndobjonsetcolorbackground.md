# OnSetColorBackground

The virtual handler of control "SetColorBackground" (change of the [OBJPROP_BGCOLOR](/en/docs/constants/objectconstants/enum_object_property#enum_object_property_integer) property) event.

```
virtual bool  OnSetColorBackground()

```

Return Value

true - event processed, otherwise - false.

Note

The base class method does nothing and always returns true.
