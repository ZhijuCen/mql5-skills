# SenkouSpanB

Returns the buffer element of the SenkouSpanB line by the specified index.

```
double  SenkouSpanB(
   int  index      // index
   )

```

Parameters

index

[in]  SenkouSpanB line buffer element index.

Return Value

The buffer element of the SenkouSpanB line of the specified index, or [EMPTY_VALUE](/en/docs/constants/namedconstants/otherconstants) if there is no correct data.
