# Text (Get method)

Gets text of the label associated with the control.

```
string  Text()

```

Return Value

Text of the label.

# Text (Set method)

Sets text of the label associated with the control.

```
bool  Text(
   const string  value      // text
   )

```

Parameters

value

[in]  New text of the label.

Return Value

true - successful, otherwise - false.

Note

Label text is specified by setting [OBJPROP_TEXT](/en/docs/constants/objectconstants/enum_object_property#enum_object_property_string) (text) of a chart object.
