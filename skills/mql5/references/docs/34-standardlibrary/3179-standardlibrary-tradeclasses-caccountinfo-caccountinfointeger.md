# InfoInteger

Gets the value of specified integer type property.

```
long  InfoInteger(
   ENUM_ACCOUNT_INFO_INTEGER  prop_id     // property ID
   ) const

```

Parameters

prop_id

[in]  Identifier of the property. The value can be one of the values of [ENUM_ACCOUNT_INFO_INTEGER](/en/docs/constants/environment_state/accountinformation#enum_account_info_integer) enumeration.

Return Value

Value of [long](/en/docs/basis/types/integer/integertypes) type.
