# FontSet

Sets the current font parameters.

```
bool  FontSet(
   const string  name,        // name
   const int     size,        // size
   const uint    flags=0,     // flags
   const uint    angle=0      // angle
   )

```

Parameters

name

[in]  Name.

size

[in]  Size.

flags=0

[in]  Flags.

angle=0

[in]  Angle.

Return Value

true - successful, otherwise - false.
