# OnChartEventProcess

Sets a flag to handle the [OnChartEvent](/en/docs/event_handlers/onchartevent) event.

```
void  OnChartEventProcess(
   bool     value        // flag
   )

```

Parameters

value

[in]  Flag to handle the [OnChartEvent](/en/docs/event_handlers/onchartevent) event.

Return Value

None.

Note

If the flag is true, the [OnChartEvent](/en/docs/event_handlers/onchartevent) event is handled. By default, the flag is set to false.
