# TypeFilling

Gets the order filling type.

```
ENUM_ORDER_TYPE_FILLING  TypeFilling() const

```

Return Value

Order filling type from [ENUM_ORDER_TYPE_FILLING](/en/docs/constants/tradingconstants/orderproperties#enum_order_type_filling) enumeration.

Note

The order should be selected using the [Select](/en/docs/standardlibrary/tradeclasses/corderinfo/corderinfoselect) (by ticket) or [SelectByIndex](/en/docs/standardlibrary/tradeclasses/corderinfo/corderinfoselectbyindex) (by index) methods.
