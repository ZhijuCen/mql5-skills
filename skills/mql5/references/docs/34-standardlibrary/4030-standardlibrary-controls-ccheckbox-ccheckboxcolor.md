# Color (Get method)

Gets color of the label associated with the control.

```
color  Color()  const

```

Return Value

Label color.

# Color (Set method)

Sets color of the label associated with the control.

```
bool  Color(
   const color  value      // color
   )

```

Parameters

value

[in]  New label color.

Return Value

true - successful, otherwise - false.

Note

Label color is specified by setting [OBJPROP_COLOR](/en/docs/constants/objectconstants/enum_object_property#enum_object_property_integer) (color) of a chart object.
