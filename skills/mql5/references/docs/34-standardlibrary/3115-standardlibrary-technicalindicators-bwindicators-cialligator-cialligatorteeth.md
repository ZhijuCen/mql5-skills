# Teeth

Returns the buffer element of the Teeth line by the specified index.

```
double  Teeth(
   int  index      // index
   )

```

Parameters

index

[in]  Teeth line buffer element index.

Return Value

The buffer element of the Teeth line by the specified index, or [EMPTY_VALUE](/en/docs/constants/namedconstants/otherconstants) if there is no correct data.
