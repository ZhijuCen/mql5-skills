# TradeExecution

Gets the trade execution mode.

```
ENUM_SYMBOL_TRADE_EXECUTION  TradeExecution() const

```

Return Value

Trade execution mode from [ENUM_SYMBOL_TRADE_EXECUTION](/en/docs/constants/environment_state/marketinfoconstants#enum_symbol_trade_execution) enumeration.

Note

The symbol should be selected by [Name](/en/docs/standardlibrary/tradeclasses/csymbolinfo/csymbolinfoname) method.
