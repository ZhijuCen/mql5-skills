# Plus

Returns the buffer element of the +DI line by the specified index.

```
double  Plus(
   int  index      // index
   )

```

Parameters

index

[in]  +DI line buffer element index.

Return Value

The buffer element of the +DI line by the specified index, or [EMPTY_VALUE](/en/docs/constants/namedconstants/otherconstants) if there is no correct data.
