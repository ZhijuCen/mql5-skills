# AutoScale (Get method)

Returns the flag defining the need for auto-scale.

```
bool  AutoScale()

```

Return Value

The flag value.

Note

true — perform auto-scaling.

false — do not perform auto-scaling.

Sets the flag defining the need for auto-scale.

```
   const bool  auto      // flag value
   )

```

Parameters

auto

[in]

Note

true — perform auto-scaling.
