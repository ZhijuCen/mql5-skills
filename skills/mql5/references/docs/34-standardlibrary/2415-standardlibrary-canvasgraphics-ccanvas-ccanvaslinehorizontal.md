# LineHorizontal

Draws a segment of a horizontal line.

```
void  LineHorizontal(
   int         x1,      // X coordinate
   int         x2,      // X coordinate
   int         y,       // Y coordinate
   const uint  clr      // color
   );

```

Parameters

x1

[in]  X coordinate of the segment's first point.

x2

[in]  X coordinate of the segment's second point.

y

[in]  Segment's Y coordinate.

clr

[in]  Color in ARGB format.
