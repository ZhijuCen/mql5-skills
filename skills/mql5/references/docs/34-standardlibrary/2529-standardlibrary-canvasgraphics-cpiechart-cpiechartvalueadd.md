# ValueAdd

Adds a new value to the pie chart (to the end).

```
 bool  ValueAdd(
   const double  value,  // value
   const string  descr,  // label
   const uint    clr,    // color
   )

```

Parameters

value

[in] Value.

descr

[in] Value label.

clr

[in] Value color.

Return Value

true if successful, otherwise — false.
