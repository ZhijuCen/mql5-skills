# Lips

Returns the buffer element of the Lips line by the specified index.

```
double  Lips(
   int  index      // index
   )

```

Parameters

index

[in] Lips line buffer element index.

Return Value

The buffer element of the Lips line by the specified index, or [EMPTY_VALUE](/en/docs/constants/namedconstants/otherconstants) if there is no correct data.
