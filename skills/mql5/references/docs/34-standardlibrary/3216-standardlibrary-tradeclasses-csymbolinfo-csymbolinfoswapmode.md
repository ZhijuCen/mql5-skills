# SwapMode

Gets the swap calculation mode.

```
ENUM_SYMBOL_SWAP_MODE  SwapMode() const

```

Return Value

Swap calculation mode from [ENUM_SYMBOL_SWAP_MODE](/en/docs/constants/environment_state/marketinfoconstants#enum_symbol_swap_mode) enumeration.

Note

The symbol should be selected by [Name](/en/docs/standardlibrary/tradeclasses/csymbolinfo/csymbolinfoname) method.
