# Refresh

Updates the indicator data. It is recommended calling the method before using [GetData()](/en/docs/standardlibrary/technicalindicators/cindicators/cindicator/cindicatorgetdata).

```
virtual void  Refresh(
   int  flags=OBJ_ALL_PERIODS      // flags
   )

```

Parameters

flags=OBJ_ALL_PERIODS
