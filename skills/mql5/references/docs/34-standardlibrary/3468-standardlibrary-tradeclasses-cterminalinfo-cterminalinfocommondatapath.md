# CommonDataPath

Gets the common data folder of all client terminals installed on the computer.

```
string  CommonDataPath() const 

```

Return Value

Common data folder of all installed terminals.

Note

The common data folder of all installed terminals is defined by [TerminalInfoString()](/en/docs/check/terminalinfostring) function ([COMMON_DATA_PATH](/en/docs/constants/environment_state/terminalstatus#enum_terminal_info_integer) property).
