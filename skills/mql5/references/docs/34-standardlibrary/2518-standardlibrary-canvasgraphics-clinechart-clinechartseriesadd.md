# SeriesAdd

Adds a new data series.

```
 bool  SeriesAdd(
   const double&  value[],  // values
   const string  descr,     // label
   const uint    clr,       // color
   )

```

Parameters

value[]

[in] Dara series.

descr

[in] Series label.

clr

[in] Series display color.

Return Value

true if successful, otherwise — false.
