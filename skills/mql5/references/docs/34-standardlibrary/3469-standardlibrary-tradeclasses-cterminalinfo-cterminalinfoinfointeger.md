# InfoInteger

Returns the value of a corresponding integer property of the mql5 program environment.

```
int  TerminalInfoInteger(
   int  property_id      // identifier of a property
   );

```

Parameters

property_id

[in] Identifier of a property. It can be one of the values of the enumeration [ENUM_TERMINAL_INFO_INTEGER](/en/docs/constants/environment_state/terminalstatus#enum_terminal_info_integer).

Return Value

Value of int type.

Note

The property value is defined by [TerminalInfoInteger()](/en/docs/check/terminalinfointeger) function.
