# IsX64

Gets the information about the type of the client terminal.

```
bool  IsX64() const 

```

Return Value

true - 64-bit version is used, otherwise - false.

Note

The type of the terminal is defined by [TerminalInfoInteger()](/en/docs/check/terminalinfointeger) function ([TERMINAL_X64](/en/docs/constants/environment_state/terminalstatus#enum_terminal_info_integer) property).
