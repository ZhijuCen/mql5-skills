# PixelSetAA

Draws a point using antialiasing algorithm.

```
void  PixelSetAA(
   const double  x,       // X coordinate
   const double  y,       // Y coordinate
   const uint    clr      // color
   );

```

Parameters

x

[in]  Point's X coordinate.

y

[in]  Point's Y coordinate.

clr

[in]  Color in ARGB format.
