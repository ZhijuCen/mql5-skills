# Jaw

Returns the buffer element of the Jaws line by the specified index.

```
double  Jaw(
   int  index      // index
   )

```

Parameters

index

[in]  Jaws line buffer element index.

Return Value

The buffer element of the Jaws line by the specified index, or [EMPTY_VALUE](/en/docs/constants/namedconstants/otherconstants) if there is no correct data.
