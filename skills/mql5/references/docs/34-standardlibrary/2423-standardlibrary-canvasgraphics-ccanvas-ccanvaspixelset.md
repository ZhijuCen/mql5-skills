# PixelSet

Sets color of the point with the specified coordinates.

```
void  PixelSet(
   const int   x,       // X coordinate
   const int   y,       // Y coordinate
   const uint  clr      // color
   );

```

Parameters

x

[in]  Point's X coordinate.

y

[in]  Point's Y coordinate.

clr

[in]  Color in ARGB format.
