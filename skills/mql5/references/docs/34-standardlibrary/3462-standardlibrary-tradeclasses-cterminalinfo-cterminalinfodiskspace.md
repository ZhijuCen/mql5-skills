# DiskSpace

Gets the information about free disk space available for MQL5\Files folder of the terminal/agent (in Mb).

```
int  MDiskSpace() const 

```

Return Value

Free disk space.

Note

Free disk space is defined by [TerminalInfoInteger()](/en/docs/check/terminalinfointeger) function ([TERMINAL_DISK_SPACE](/en/docs/constants/environment_state/terminalstatus#enum_terminal_info_integer) property).
