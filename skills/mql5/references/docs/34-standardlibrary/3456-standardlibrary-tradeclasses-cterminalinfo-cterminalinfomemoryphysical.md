# MemoryPhysical

Gets the information about the physical memory (in Mb).

```
int  MemoryPhysical() const 

```

Return Value

Physical memory (in Mb).

Note

The physical memory is defined by [TerminalInfoInteger()](/en/docs/check/terminalinfointeger) function ([TERMINAL_MEMORY_PHYSICAL](/en/docs/constants/environment_state/terminalstatus#enum_terminal_info_integer) property).
