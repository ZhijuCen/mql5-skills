# Lower

Returns the buffer element of the lower line by the specified index.

```
double  Lower(
   int  index      // index
   )

```

Parameters

index

[in]  Lower line buffer element index.

Return Value

The buffer element of the lower line by the specified index, or [EMPTY_VALUE](/en/docs/constants/namedconstants/otherconstants) if there is no correct data.
