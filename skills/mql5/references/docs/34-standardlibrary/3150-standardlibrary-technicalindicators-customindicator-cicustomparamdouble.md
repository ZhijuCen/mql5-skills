# ParamDouble

Gets the value of specified parameter of double type.

```
double  ParamDouble(
   int  index      // index
   ) const

```

Parameters

index

[in]  Parameter index.

Return Value

The value of specified parameter of double type, used in creation of the indicator.

Note

If the parameter index is invalid or the parameter type is not of double type, it returns [EMPTY_VALUE](/en/docs/constants/namedconstants/otherconstants).
