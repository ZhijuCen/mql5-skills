# Polyline

Draws a polyline.

```
void  Polyline(
   int&        x[],     // array of X coordinates
   int&        y[],     // array of Y coordinates
   const uint  clr      // color
   );

```

Parameters

x[]

[in]  Array of X coordinates of a polyline.

y[]

[in]  Array of Y coordinates of a polyline.

clr

[in]  Color in ARGB format.
