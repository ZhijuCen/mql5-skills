# FontGet

Gets the current font parameters.

```
void  FontGet(
   string  &name,      // name
   int     &size,      // size
   uint    &flags,     // flags
   uint    &angle      // angle
   )

```

Parameters

&name

[out]  Name.

&size

[out]  Size.

&flags

[out]  Flags.

&angle

[out]  Angle.
