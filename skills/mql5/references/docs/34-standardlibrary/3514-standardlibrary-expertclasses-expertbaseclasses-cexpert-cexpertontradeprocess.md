# OnTradeProcess

Sets the [OnTrade](/en/docs/event_handlers/ontrade) event handling flag.

```
void  OnTradeProcess(
   bool     value        // flag
   )

```

Parameters

value

[in]  [OnTrade](/en/docs/event_handlers/ontrade) event handling flag.

Return Value

None.

Note

If the flag is true, the [OnTrade](/en/docs/event_handlers/ontrade) event is handled. By default, the flag is set to false.
