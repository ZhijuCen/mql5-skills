# OnSetFontSize

The virtual handler of "SetFontSize" (change of the [OBJPROP_FONTSIZE](/en/docs/constants/objectconstants/enum_object_property#enum_object_property_integer) property) event.

```
virtual bool  OnSetFontSize()

```

Return Value

true - event processed, otherwise - false.

Note

The base class method does nothing and always returns true.
