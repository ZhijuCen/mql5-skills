# CodePage

Gets the information about code page of the language in the terminal.

```
int  CodePage() const 

```

Return Value

Code page of the language in the client terminal.

Note

Code page of the language is defined by [TerminalInfoInteger()](/en/docs/check/terminalinfointeger) function ([TERMINAL_CODEPAGE](/en/docs/constants/environment_state/terminalstatus#enum_terminal_info_integer) property).
