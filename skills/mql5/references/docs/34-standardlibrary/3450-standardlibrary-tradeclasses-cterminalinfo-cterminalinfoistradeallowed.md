# IsTradeAllowed

Gets the information about permission to trade.

```
bool  IsTradeAllowed() const 

```

Return Value

true - trade allowed, otherwise - false.

Note

Permission to trade is defined by [TerminalInfoInteger()](/en/docs/check/terminalinfointeger) function ([TERMINAL_TRADE_ALLOWED](/en/docs/constants/environment_state/terminalstatus#enum_terminal_info_integer) property).
