# TradeCalcMode

Gets the mode of contract cost calculation.

```
ENUM_SYMBOL_CALC_MODE  TradeCalcMode() const

```

Return Value

Mode of contract cost calculation from [ENUM_SYMBOL_CALC_MODE](/en/docs/constants/environment_state/marketinfoconstants#enum_symbol_calc_mode) enumeration.

Note

The symbol should be selected by [Name](/en/docs/standardlibrary/tradeclasses/csymbolinfo/csymbolinfoname) method.
