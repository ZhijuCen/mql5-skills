# OnTimerProcess

Sets the [OnTimer](/en/docs/event_handlers/ontimer) event handling flag.

```
void  OnTimerProcess(
   bool     value        // flag
   )

```

Parameters

value

[in]  [OnTimer](/en/docs/event_handlers/ontimer) event handling flag.

Return Value

None.

Note

If the flag is true, the [OnTimer](/en/docs/event_handlers/ontimer) event is handled. By default, the flag is set to false.
