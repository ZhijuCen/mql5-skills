# SenkouSpanA

Returns the buffer element of the SenkouSpanA line by the specified index.

```
double  SenkouSpanA(
   int  index      // index
   )

```

Parameters

index

[in]  SenkouSpanA line buffer element index.

Return Value

The buffer element of the SenkouSpanA line of the specified index, or [EMPTY_VALUE](/en/docs/constants/namedconstants/otherconstants) if there is no correct data.
