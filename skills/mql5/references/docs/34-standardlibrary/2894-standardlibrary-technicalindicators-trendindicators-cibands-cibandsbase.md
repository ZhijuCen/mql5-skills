# Base

Returns the buffer element of the base line by the specified index.

```
double  Base(
   int  index      // index
   )

```

Parameters

index

[in]  Base line buffer element index.

Return Value

The buffer element of the base line by the specified index, or [EMPTY_VALUE](/en/docs/constants/namedconstants/otherconstants) if there is no correct data.
