# OnTickProcess

Sets the [OnTick](/en/docs/event_handlers/ontick) event handling flag.

```
void  OnTickOProcess(
   bool     value        // flag
   )

```

Parameters

value

[in]  [OnTick](/en/docs/event_handlers/ontick) event handling flag.

Return Value

None.

Note

If the flag is true, the [OnTick](/en/docs/event_handlers/ontick) event is handled. By default, the flag is set to true.
