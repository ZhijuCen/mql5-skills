The following identifiers are recorded as reserved words, each of them corresponds to a certain action, and cannot be used in another meaning:

| char | int | ulong |
| class | long | union |
| color | short | ushort |
| datetime | string | void |
| double | struct |  |
| enum | uchar |  |
