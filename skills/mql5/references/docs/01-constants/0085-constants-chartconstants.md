# Chart Constants

Constants describing various properties of charts are divided into the following groups:

- [Types of events](/en/docs/constants/chartconstants/enum_chartevents) – events that occur when working with charts;
- [Chart timeframes](/en/docs/constants/chartconstants/enum_timeframes) – standard built-in periods;
- [Properties of chart](/en/docs/constants/chartconstants/enum_chart_property) – identifiers that are used as parameters of [chart functions](/en/docs/chart_operations);
- [Positioning constants](/en/docs/constants/chartconstants/enum_chart_position) - value of a parameter of the [ChartNavigate()](/en/docs/chart_operations/chartnavigate) function;
- [Displaying charts](/en/docs/constants/chartconstants/chart_view) - setting the chart appearance.
