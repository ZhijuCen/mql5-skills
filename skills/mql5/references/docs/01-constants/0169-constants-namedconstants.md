# Named Constants

All constants used in MQL5 can be divided into the following groups:

- [Predefined macro substitutions](/en/docs/constants/namedconstants/compilemacros) – values are substituted during compilation;
- [Mathematical constants](/en/docs/constants/namedconstants/mathsconstants) – values of some mathematical expressions;
- [Numerical type constants](/en/docs/constants/namedconstants/typeconstants) –  some of the simple type restrictions;
- [Uninitialization reason codes](/en/docs/constants/namedconstants/uninit) – description of uninitialization reasons;
- [Checking Object Pointer](/en/docs/constants/namedconstants/enum_pointer_type) – enumeration of types of pointers returned by the [CheckPointer()](/en/docs/common/checkpointer) function;
- [Other constants](/en/docs/constants/namedconstants/otherconstants) – all other constants.
