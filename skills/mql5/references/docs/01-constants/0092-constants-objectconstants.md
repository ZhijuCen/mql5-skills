# Object Constants

There are 44 graphical objects that can be created and displayed in the price chart. All constants for working with objects are divided into 9 groups:

- [Object types](/en/docs/constants/objectconstants/enum_object) – Identifiers of graphical objects;
- [Object properties](/en/docs/constants/objectconstants/enum_object_property) – setting and getting properties of graphical objects;
- [Methods of object binding](/en/docs/constants/objectconstants/enum_anchorpoint) – constants of object positioning in the chart;
- [Binding corner](/en/docs/constants/objectconstants/enum_basecorner) – setting the corner relative to which an object is positioned on chart;
- [Visibility of objects](/en/docs/constants/objectconstants/visible) – setting timeframes in which an object is visible;
- [Levels of Elliott Waves](/en/docs/constants/objectconstants/enum_elliot_wave_degree) – gradation of waves;
- [Gann objects](/en/docs/constants/objectconstants/enum_gann_direction) – trend constants for Gann fan and Gann grid;
- [Web colors](/en/docs/constants/objectconstants/webcolors) – constants of predefined web colors;
- [Wingdings](/en/docs/constants/objectconstants/wingdings) – codes of characters of the Wingdings font.
