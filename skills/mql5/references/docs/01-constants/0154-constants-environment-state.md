# Environment State

Constants describing the current runtime environment of an mql5-program are divided into groups:

- [Client terminal properties](/en/docs/constants/environment_state/terminalstatus) – information about the client terminal;
- [Executed MQL5-program properties](/en/docs/constants/environment_state/mql5_programm_info) –  mql5 program properties, which help to control its execution;
- [Symbol properties](/en/docs/constants/environment_state/marketinfoconstants) – obtaining information about a symbol;
- [Account properties](/en/docs/constants/environment_state/accountinformation)  – information about the current account;

- [Testing Statistics](/en/docs/constants/environment_state/statistics) – results of Expert Advisor testing.
